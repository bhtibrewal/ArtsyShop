### Features:
- Authentication
- Add to cart/ Remove from Cart
- Add to wishlist/ Remove from wishlist
- Update Cart quantity
- Filter based on Category, Rating, Availibilty, Fast delivery
- Sort on the basis of Price
- Search a product
- Address Management
- Private Routing
- Toasts for success/ error
- Keep user logged in
- Apply Coupons
- Responsive Sidebar
- Dark-Light theme

### Pages:
- Login/ Signup Page
- Home Page
- Product Listing Page
- Wishlist Page
- Cart Page
- Single Product Page
- User-profile Page
- Checkout Page
- 404 Page
